This will guide you through the the contents of this folder & structure


Required data to run the code is available in the 'Data' folder
1. Data Preprocessing -> In 'Preprocessing' folder. For converting json files into csv.
2. Satisfaction Prediction -> Satisfaction Prediction folder contains Implementation in R.
3. Descriptive Statistics and Visualization -> Descriptive Statistics and images of visualizations created
4. User Based Collaborative Filtering -> This folder contains the R code to run the collaborative Filtering Algorithm.




NOTE: The outputs of the R code are included in the .html files. The code may take a long time to run due to size of datasets. Additionally, the SVM Model takes a large amount of time to complete execution
